HTML

  • The head element contains information about the webpage for the browser.
  • The head element contains metadata about the character set and viewport.
  • The head element also contains the website title, which is displayed on the browser tab. This is usually the only child of the head element that is visible to the user.
  • The body element represents the visible content shown to the user.
  • The body element can include text, images and hyperlinks.
  • The body element can also contain scripts to make the page interactive.
  • Semantic HTML elements are preferred because they help users with screen readers understand the page structure and the purpose of each element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
